Definitions

Syntax:

Aspects of a programming language that can be modeled by a context free grammar.

Semantics:

Rules of a programming language that help us interpret the meaning of a given string.

Lexical Scanning:

Converting sequences of single character tokens to a set of units that have their own meaning.

Parsing & Parser:

A compiler or interpreter component that breaks data into smaller elements for easy translation into another language. A parser takes input in the form of a sequence of tokens, interactive commands, or program instructions and breaks them up into parts that can be used by other components in programming.
